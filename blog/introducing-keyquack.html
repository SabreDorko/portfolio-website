keyQuack is a productivity desktop pet that tracks your keyboard activity and rewards active use through a fishing minigame. It's currently in beta for Windows — free to download, with an optional "Buy me a Byte" support option.

This is officially a beta as the game is still in development, so expect some bugs and things to change.

How It Works

keyQuack lives on your desktop while you work. As you type, a gauge fills up. When it's full, you can send your duck diving — and while your duck is underwater, typing will speed up their progress. Fish you catch get catalogued in a collection accessible via the bucket icon.

keyQuack uses Python for global input detection while running in the background. To be clear: keyQuack does not track what you are typing, nor is this data sent anywhere. Your typing activity is saved locally.

Features

Dashboard

Accessible via the duck button, the dashboard contains your stats, tasks, and timer tools. Stats update every second and pause after one minute of inactivity.

Tasks

A built-in to-do list where items can be added and marked complete, then cleared. Empty tasks auto-delete to keep things tidy.

Timer

Customizable timers that appear in movable popup windows, always on top of other applications — useful for keeping track of work sessions or breaks.

Fishing Minigame

The core loop. Typing fills a gauge, and when it's full, you send your duck fishing. The fish you catch are catalogued in a collection you can browse anytime.

Made With

keyQuack was built with Godot and Clip Studio Paint. No generative AI was used in its creation.
