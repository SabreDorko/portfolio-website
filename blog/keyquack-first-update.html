After having uploaded the first installer for keyQuack, I was made aware of two major issues:

  • Progress towards dives does not save between sessions, making it frustrating to get close to a dive and then close out the program only to start from scratch
  • Dives take a long time to progress
  • Users would like an hours field in the timer setup

So, I've made some adjustments.

Changes

  • Previous installer has been replaced with a simple .zip file
  • Stats now update once every second
  • AFK timer is now more accurate, starting as soon as you stop typing and adjusting stats accordingly
  • Total Active Time now shows seconds as well
  • Fixed the select sound playing on startup if user had a completed task in their tasks list
  • Cost to fill the gauge for being able to dive has been drastically reduced from the original 2,400 keystrokes to 750
  • Gauge will now save progress every few seconds, so closing the program won't set you back to 0
  • Timer has been converted from a spinbox to text fields and an Hours tab has been added
